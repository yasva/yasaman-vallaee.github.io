Conway’s Game of Life

Console-based implementation of Conway’s Game of Life developed as the final project for the course Programming Fundamentals (DT179G) at Mid Sweden University.

The project demonstrates core programming concepts such as data structures, control flow, modular design, and algorithmic thinking. The simulation follows Conway’s rules for cellular automata and runs step-by-step in the terminal.

The implementation was extended incrementally to meet higher grade requirements, including recursive simulation, loading initial states from file, status logging, and aging of cells.

Tech: Python, CLI, Algorithms, Data Structures

Course: DT179G – Programming Fundamentals (MIUN)
